---
title: REST API
---

Checkmate's REST API was designed to access and automate various aspects of Checkmate's project structure to implement custom scripts and automation integrations. We designed and implemented the REST API to support most of the use cases.

[Postman](https://www.postman.com/checkmate-2397/checkmate/collection/504lfpr/checkmate-api-endpoints-collection) collection of APIs is currently available, comprehensive documentation is in progress.
